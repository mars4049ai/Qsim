АННОТАЦИЯ ПРОГРАММНОЙ БИБЛИОТЕКИ Qsim

Область: квантовые вычисления, квантовые коммуникации, квантовая криптография.

Назначение

Библиотека Qsim предназначена для симуляции процессов квантового распределения
ключей (QKD), квантовой памяти и защиты данных от атак типа "человек посередине"
(MITM). Основная цель -- предоставить исследовательский и учебный инструментарий,
позволяющий воспроизвести ключевые физические явления квантовой информатики в
программной среде без использования реального квантового оборудования.

Методы и принципы реализации

В основе библиотеки лежат три группы методов:

1. Протокол BB84 (Bennett--Brassard, 1984).
   Квантовое распределение ключей реализовано на базе симулятора квантовых схем
   Qiskit Aer. Алиса готовит кубиты в случайных базисах (вычислительном и
   Адамара), Боб производит измерения в независимо выбранных базисах. При наличии
   перехватчика (Евы) её измерения вносят ошибки в согласованный ключ, что
   выражается в повышении квантового уровня битовых ошибок (QBER). Безопасный
   канал считается установленным при QBER ниже порогового значения (по умолчанию
   10%). Шифрование передаваемых сообщений осуществляется алгоритмом XOR с
   ключом, полученным через SHA-256-хеширование квантового ключа.

2. Квантовая память на кудитах.
   Хранение данных моделируется через состояния Фока квантовых гармонических
   осцилляторов (QuTiP). Каждый байт входных данных кодируется несколькими
   кудитами -- цифрами в системе счисления с основанием, равным числу уровней
   осциллятора. Класс QuantumMemoryServer реализует полный цикл записи и чтения
   зашифрованных данных клиентов с контролем целостности.

3. Деградация квантовой памяти (уравнение Линдблада).
   Временная эволюция квантового состояния моделируется через мастер-уравнение
   Линдблада--Горини--Косаковски--Сударшана с оператором распада (оператор
   уничтожения фотонов). Реализованы три режима: без защиты (скорость затухания
   kappa), с защитным кодированием на основе кодов Бозе (kappa/15) и идеальная
   память. Сравнительный анализ режимов отображается через графики сохранности
   данных и GIF-анимации 2D-динамики вероятностной плотности.

Состав библиотеки

   Qsim                  -- главный класс QKD-симулятора и шифрования.
   QuantumMemoryServer   -- сервер квантовой памяти на кудитах.
   RealisticQuantumMemory-- модель деградации квантовой памяти.

Практическая значимость

Библиотека может быть использована для:
   - демонстрации принципов квантовой криптографии в учебных курсах;
   - исследования влияния декогеренции на надёжность квантовой памяти;
   - визуализации квантовой динамики для научных публикаций;
   - прототипирования протоколов квантовой сети.

Зависимости: Python >= 3.9, NumPy, SciPy, Matplotlib, QuTiP >= 5.0,
Qiskit >= 1.0, Qiskit-Aer >= 0.14.

Ключевые слова: квантовое распределение ключей, BB84, QBER, квантовая память,
кудиты, уравнение Линдблада, декогеренция, квантовая криптография, MITM-атака,
гармонический осциллятор, QuTiP, Qiskit.
